options folder modifies:
precache - whether to precache the sprites or not (use true or false)
sound - Turn sound on or off (use true or false)
volume - adjust volume (multiplier, use numbers)
useTickSound - whether to allow the keyboard tallying noise when xCombo is visible (use true or false)
showComboNumbers - Turn the regular combo numbers on or off (use true or false)
noteComboGlowAnimation - Turn the special note comboGlow animation and combo Splashes on or off, replaces the current texture (use true or false)
noteShineAnimation - Turn the notes shining once the combo dissappears on or off (use true or false)
noteShineTransparency - adjust the brightness of the shine (use numbers from 1 to 0)
centerXcombo - whether to center the xCombo graphic (use true or false)
totalComboTally - tally Exclusive, count up the total combo instead of the current combo (use true or false)
updatePosition - whether to constantly update the position of the graphics (use true or false)
lightningOnSustains - flash specific, whether to play lightning bolts on every sustain note (use true or false) 
randomizedLightningSounds - flash specific, whether to randomize lightning sounds (use true or false)

default settings:
precache = true
sound = true
volume = 1
useTickSound = false
showComboNumbers = false
noteShineAnimation = true
noteShineTransparency = 1
centerXcombo = false
updatePosition = true
totalComboTally = false
lightningOnSustains = true
randomizedLightningSounds = true
